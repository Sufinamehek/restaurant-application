# My Awesome Project
A modern, responsive web application built with React and Vite. This project showcases a restaurant menu, ordering flow, and responsive design.

## Features
- Fully responsive design for mobile and desktop
- Interactive menu and cart functionality
- Hero image slider with smooth transitions
- Contact and About pages
- Deployment-ready on Vercel

## Technologies Used
- React
- Vite
- Tailwind CSS / CSS
- React Router
- Vercel for deployment
